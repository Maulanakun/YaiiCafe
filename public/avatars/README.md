# Avatar 3D Models

Folder ini berisi file-file model 3D dalam format GLB (glTF Binary) untuk avatar Roblox.

## File yang Diperlukan

Untuk menampilkan avatar 3D di explore page, tambahkan file-file berikut:

- `knight.glb` - Avatar Royal Knight
- `explorer.glb` - Avatar Galaxy Explorer
- `princess.glb` - Avatar Fairy Princess
- `geek.glb` - Avatar Tech Geek
- `popstar.glb` - Avatar Pop Star
- `casual.glb` - Avatar Casual Gamer

## Cara Mendapatkan Model

1. **Export dari Roblox Studio**
   - Pilih model/character di Roblox Studio
   - Klik kanan → Export Selection
   - Simpan sebagai format `.rbxm` atau `.obj`
   - Konversi ke GLB menggunakan tools online atau Blender

2. **Dari Website Pihak Ketiga**
   - Kunjungi website 3D model repository
   - Download model humanoid/avatar
   - Konversi ke format GLB jika perlu

3. **Menggunakan Blender**
   - Import model (OBJ, FBX, atau format lain)
   - Beri nama dan setup material
   - Export sebagai glTF Binary (.glb)

## Format File

- Format: GLB (glTF Binary)
- Ukuran optimal: 5-15 MB per file
- Texture: Baked atau embedded dalam GLB

## Testing

Jika tidak memiliki file GLB asli, aplikasi akan menampilkan pesan error yang user-friendly.
Ganti URL path `/avatars/knight.glb` dengan path model GLB yang sebenarnya.
